Depth First Search (DFS)

Definition

Depth First Search (DFS) is a graph traversal algorithm that explores as far as possible along each branch before backtracking. It starts from a source node and explores deeper nodes first before moving to adjacent nodes.

Why Stack is Used in DFS?

  • DFS follows Last In First Out (LIFO) principle.
  • Stack helps to keep track of nodes to be visited next.
  • It supports backtracking when a node has no unvisited neighbors.
  • DFS can be implemented using recursion (implicit stack) or explicit stack.

Procedure to Traverse Graph using DFS

  1. Start from the root node.
  2. Mark the node as visited.
  3. Visit any unvisited adjacent node.
  4. Repeat steps recursively until no unvisited nodes remain.
  5. Backtrack to previous node and continue traversal.

Advantages of DFS in Biotechnology

  • Used in genome sequencing to explore possible gene paths.
  • Helps in protein interaction network analysis.
  • Useful in phylogenetic tree construction.
  • Supports pathway analysis in metabolic networks.
