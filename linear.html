Linear Probing in Hashing

Definition

Linear Probing is an open addressing collision resolution technique in hashing. When a collision occurs, the algorithm checks the next consecutive slot (i+1, i+2, i+3 â€¦) until an empty slot is found.

Algorithm (Insertion using Linear Probing)

  1. Start
  2. Read the key element k to be inserted
  3. Compute hash index using hash function index = h(k)
  4. If hashTable[index] is empty, insert element there
  5. If occupied, move to next slot index = (index + 1) % table_size
  6. Repeat step 4 until empty slot found
  7. Insert the element
  8. Stop

C Implementation using Array

#include <stdio.h>
#define SIZE 10

int hashTable[SIZE];

void init(){
    for(int i=0;i<SIZE;i++)
        hashTable[i] = -1;
}

void insert(int key){
    int index = key % SIZE;

    while(hashTable[index] != -1){
        index = (index + 1) % SIZE;
    }

    hashTable[index] = key;
}

void display(){
    printf("Hash Table:\n");
    for(int i=0;i<SIZE;i++){
        printf("%d --> %d\n", i, hashTable[i]);
    }
}

int main(){
    int elements[10] = {23,43,13,27,33,53,63,73,83,93};
    init();

    for(int i=0;i<10;i++){
        insert(elements[i]);
    }

    display();
    return 0;
}
