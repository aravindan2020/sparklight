Expression Tree

Definition

An Expression Tree is a binary tree used to represent arithmetic expressions. In this tree:

  • Leaf nodes represent operands (variables or constants)
  • Internal nodes represent operators (+, −, ×, ÷)
Expression trees help computers evaluate expressions and maintain the correct order of operations.

Example

Expression : A + B × C

      +
     / \
    A   *
       / \
      B   C

Algorithm to Construct Expression Tree (From Postfix)

  1. Create an empty stack.
  2. Scan the postfix expression from left to right.
  3. If the symbol is an operand, create a node and push it into the stack.
  4. If the symbol is an operator:
    • Pop two nodes from the stack.
    • Create a new node with the operator.
    • Make the first popped node as right child.
    • Make the second popped node as left child.
    • Push the new node back to the stack.
  5. Repeat until the expression ends.
  6. The final node in the stack becomes the root of the expression tree.

Purpose of Expression Tree

  • Represents mathematical expressions in tree structure.
  • Maintains operator precedence automatically.
  • Used for expression evaluation.
  • Helps convert between infix, prefix, and postfix expressions.
  • Widely used in compiler design.

Traversal Results of Expression Tree

For the expression A + B × C:

Traversal Result Meaning
Inorder A + B × C Produces the original infix expression
Preorder + A × B C Produces prefix notation
Postorder A B C × + Produces postfix notation

Why Traversals are Important

  • Inorder: Displays the normal mathematical expression.
  • Preorder: Used in prefix expression evaluation.
  • Postorder: Used in postfix evaluation and stack-based computation.

Application in Biotechnology

Expression trees can be used in bioinformatics for evaluating complex biological formulas.

  • Gene sequence pattern analysis
  • Mathematical modeling of protein interactions
  • Drug discovery simulations
  • Genomic data processing

For example, computational biology models often evaluate complex formulas representing gene expression levels or protein interaction scores using tree-based representations.
