Sorting Techniques using Biotechnology Examples

1. Bucket Sort

üìò Theory

Bucket Sort is a sorting technique where elements are first distributed into several groups called buckets.

Each bucket contains elements of a specific range, and sorting is done inside each bucket before combining the results.

  • Divide data into value ranges
  • Place elements into corresponding buckets
  • Sort individual buckets
  • Merge all buckets to get the final result

üß¨ Biotechnology Example

Consider gene expression levels collected from biological samples. The values range between 0.0 and 1.0.

Gene Expression Values:

0.12, 0.78, 0.34, 0.56, 0.91, 0.25

üîç Step-by-Step Working

Step 1: Create buckets based on value range
Bucket 0 ‚Üí 0.0 ‚Äì 0.2
Bucket 1 ‚Üí 0.2 ‚Äì 0.4
Bucket 2 ‚Üí 0.4 ‚Äì 0.6
Bucket 3 ‚Üí 0.6 ‚Äì 0.8
Bucket 4 ‚Üí 0.8 ‚Äì 1.0
Step 2: Place values into buckets
Bucket 0 ‚Üí 0.12
Bucket 1 ‚Üí 0.25, 0.34
Bucket 2 ‚Üí 0.56
Bucket 3 ‚Üí 0.78
Bucket 4 ‚Üí 0.91
Step 3: Sort each bucket individually
Step 4: Combine all buckets
Final Sorted Output: 0.12, 0.25, 0.34, 0.56, 0.78, 0.91

2. External Sorting

üìò Theory

External Sorting is used when the data set is too large to fit entirely into main memory.

The data is divided into smaller chunks, sorted separately, and then merged to produce a complete sorted output.

  • Divide large data into manageable blocks
  • Sort each block in memory
  • Store sorted blocks on disk
  • Merge all blocks to obtain final sorted data

üß¨ Biotechnology Example

Step 3: Save the sorted chunks back to disk
