Quadratic Probing in Hashing

Definition

Quadratic Probing is an open addressing collision resolution technique. Instead of checking consecutive slots like linear probing, it checks positions using square increments: h(k) + 1², h(k) + 2², h(k) + 3² … This reduces primary clustering.

Algorithm (Insertion using Quadratic Probing)

  1. Start
  2. Read key element k
  3. Compute hash index index = h(k)
  4. If slot empty → insert
  5. If occupied, try index = (h(k) + i²) % table_size
  6. Increase i = i + 1
  7. Repeat until empty slot found
  8. Insert element
  9. Stop

C Implementation using Array

#include <stdio.h>
#define SIZE 10

int hashTable[SIZE];

void init(){
    for(int i=0;i<SIZE;i++)
        hashTable[i] = -1;
}

void insert(int key){
    int i = 0;
    int index;

    while(i < SIZE){
        index = (key % SIZE + i*i) % SIZE;

        if(hashTable[index] == -1){
            hashTable[index] = key;
            return;
        }
        i++;
    }

    printf("Hash table full\n");
}

void display(){
    printf("Hash Table:\n");
    for(int i=0;i<SIZE;i++){
        printf("%d --> %d\n", i, hashTable[i]);
    }
}

int main(){
    int elements[10] = {23,43,13,27,33,53,63,73,83,93};

    init();

    for(int i=0;i<10;i++){
        insert(elements[i]);
    }

    display();
    return 0;
}
