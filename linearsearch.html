Linear Search

Definition

Linear Search is a simple search algorithm that sequentially checks each element in a list or array until the target value is found or the entire list has been traversed. It has a time complexity of O(n) in the worst and average cases, where n is the number of elements, and O(1) in the best case (target found at the first position). Linear Search is easy to implement and works on unsorted or sorted data but is inefficient for large datasets.

Real-World Examples

  • Searching for a contact in a small, unsorted phonebook by checking each name sequentially.
  • Finding a specific book in a small, unorganized library by examining each shelf one by one.
  • Locating an item in a grocery list by scanning each entry until the desired item is found.
  • Checking a short list of tasks to find a specific task in a to-do app without sorting.
  • Identifying a student’s name in a small, unsorted class roster by reviewing each entry.
