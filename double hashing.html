Double Hashing

Definition

Double Hashing is an open addressing collision resolution technique that uses two different hash functions. The first function gives the initial position, and the second function decides how far to jump when a collision occurs. This avoids clustering and distributes keys more uniformly.

Algorithm

  1. Start
  2. Read key element k
  3. Compute first hash h1(k) = k % table_size
  4. Compute second hash h2(k) = R − (k % R) (where R is a prime smaller than table size)
  5. If slot at h1(k) empty → insert
  6. If occupied, compute new index (h1(k) + i*h2(k)) % table_size
  7. Increase i and repeat until empty slot found
  8. Insert element
  9. Stop

C Implementation using Array

#include <stdio.h>
#define SIZE 10

int table[SIZE];

int h1(int key){
    return key % SIZE;
}

int h2(int key){
    return 7 - (key % 7);
}

void init(){
    for(int i=0;i<SIZE;i++)
        table[i] = -1;
}

void insert(int key){
    int index = h1(key);
    int step = h2(key);
    int i = 0;

    while(table[index] != -1){
        i++;
        index = (h1(key) + i*step) % SIZE;
    }

    table[index] = key;
}

void display(){
    for(int i=0;i<SIZE;i++)
        printf("%d -> %d\n", i, table[i]);
}

int main(){
    int keys[10] = {23,43,13,27,33,53,63,73,83,93};

    init();

    for(int i=0;i<10;i++)
        insert(keys[i]);

    display();
    return 0;
}
