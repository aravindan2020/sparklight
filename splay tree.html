Splay Tree

Definition

A Splay Tree is a self-adjusting binary search tree where recently accessed elements are moved to the root using a process called splaying. This improves access time for frequently used data. It performs operations like search, insert, and delete in O(log n) amortized time.

Step-by-Step Procedure (Splaying)

  • Step 1: Search the required node in the tree.
  • Step 2: Move the node to root using rotations.
  • Zig: Node is child of root → single rotation.
  • Zig-Zig: Node and parent both left/right → double rotation.
  • Zig-Zag: Node and parent opposite direction → rotation twice.
  • Step 3: After operation, accessed node becomes root.

Advantages (Biotechnology Perspective)

  • Frequently used gene/protein data can be accessed faster.
  • Useful in DNA sequence search where repeated queries occur.
  • Improves performance in medical databases with repeated patient data access.
  • Efficient for bioinformatics tools handling large datasets.
  • Requires less memory (no extra balance information).
